# Poker socket server

Websocket sever for poker bot battles. For running gcp cloud run

#### Run locally  

To test the server or run locally do: 

first fork repository.

then:
```shell
docker build -t <image name> .
```

then: 

```shell
docker run -p 4200:4200 <image name>
```
